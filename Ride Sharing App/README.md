# Ride Sharing App

A Flutter-based Ride Sharing application designed for seamless booking, real-time updates, and an intuitive user experience for Corporate World.

---

## **Getting Started**

### 1. Clone the Repository
```bash
git clone https://gitlab.com/office-mate/mobile-app.git
````

### 2. Navigate to the Project Directory

```bash
cd "Ride Sharing App"
```

### 3. Get Flutter Dependencies

Install all required packages:

```bash
flutter pub get
```

### 4. Run the App

To launch the app on an emulator or connected device:

```bash
flutter run
```
